# Todo List App

A simple browser-based Todo List app for adding, completing, and removing tasks.

## Features

- Add new tasks
- Mark tasks as completed
- Delete tasks
- Lightweight and easy to use

## Project Structure

- `index.html` – App structure
- `style.css` – App styling
- `script.js` – App behavior
- `images/` – Image assets

## How to Run

1. Open `index.html` in your browser.
2. Start adding your tasks.

## Notes

This project is built with plain HTML, CSS, and JavaScript (no framework required).
