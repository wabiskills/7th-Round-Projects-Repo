# Todo List App

A simple and responsive Todo List web app built with HTML, CSS, and JavaScript.

## Features

- Add new tasks
- Prevent empty tasks
- Mark tasks as completed
- Delete tasks individually
- Save tasks using `localStorage` (tasks remain after page refresh)
- Mobile-friendly layout

## Project Structure

- `index.html` - app structure and layout
- `style.css` - app styling and responsive design
- `script.js` - app logic (add, complete, delete, storage)

## How to Run

1. Open the `Todo-list` folder.
2. Double-click `index.html` (or open it with your browser).
3. Start adding tasks.

## How to Use

1. Type a task in the input field.
2. Click **Add Task** to add it to the list.
3. Click a task text to mark it complete/incomplete.
4. Click **Delete** to remove a task.

## Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
